# First-AI-Pandas-Notebook
This repository outlines my first experiences using the Python library Pandas. I implemented Pandas as well as Matplotlib to analyze datasets and visualize the information that each dataset held. I coded these projects in Google Colab, and each notebook was uploaded in .ipynb and .py, so that the project can also be downloaded and used on Google Colab. The first notebook I created was called AI Pandas Notebook, and it was the first project I made in my AI Pathway class. It shows how I used Pandas to analyze a variety of datasets, and additionally how I used Matplotlib to visualize these datasets. Next, in my AI pathway class, I created another project to submit as a summative. This is uploaded in the folder of this repository called "Pandas-Summative". Here I used an external dataset and performed Pandas functions and methods on the dataset to deeper analyze it. Both projects show my progression in using Pandas when analyzing datasets.
